# Project structure

- `src/qunet2/` — core package code
- `train.py`, `evaluate.py`, `predict.py`, `demo.py`, `api.py`, `main.py` — root launchers
- `assets/` — figures displayed in the repository preview
- `results/` — metrics, summaries, and exported outputs
- `configs/` — experiment and dataset configuration
- `docs/` — method, usage, deployment, and experiment notes
- `scripts/` — utility scripts
- `tests/` — automated checks
